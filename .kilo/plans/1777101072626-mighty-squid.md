# План: Carnivorous Plant Night Mortality

## Проблема

Хищные растения практически бессмертны — ночь не создаёт энергетического стресса, потому что `night_dormancy_mult = 0.3` и `food_regen_mult = 0.3` масштабируют стоимость и доход пропорционально. Конфиг `photosynth_night_mult` определён, но нигде не используется (мёртвый код).

## Решение: Разорвать пропорциональность ночь→стоимость/доход

### Параметры

| Параметр | Было | Стало | Эффект |
|----------|------|-------|--------|
| `night_dormancy_mult` | 0.3 | **0.6** | Растения платят 60% метаболизма ночью (было 30%) |
| `photosynth_night_mult` | 0.3 (не использ.) | **0.1** | Фотосинтез ночью = 10% от дневного (вместо 30% через food_regen_mult) |

### Баланс (gene[13]=0.7, лес)

| Фаза | Доход | Расход | Нетто | За фазу |
|------|-------|--------|-------|---------|
| День (660 т) | 0.728 | 0.600 | +0.128 | +84.5 |
| Ночь (360 т) | 0.073 | 0.360 | -0.287 | -103.4 |
| **Цикл** | | | | **-18.9** |

**Циклов до смерти без еды:** ~6 (~2 минуты). С ловушками — бессмертны.

| Тип | Циклов до смерти |
|-----|-----------------|
| gene[13]=0.5, land | 0.4 (мгновенно) |
| gene[13]=0.7, land | 0.9 |
| gene[13]=0.7, forest | 6.1 (~2 мин) |
| gene[13]>0.77, forest | Бессмертны (эволюционная ниша, ~23%) |

#### 2. `utils/numba_kernels.py` — использовать новые параметры

Добавить 2 параметра в `energy_update_kernel`:
```python
def energy_update_kernel(
    ...,
    photosynth, food_regen_mult,
    plant_night_dormancy,          # NEW: config.carnivorous_plant.night_dormancy_mult
    plant_photo_night_mult,        # NEW: config.carnivorous_plant.photosynth_night_mult
    _entity_count, _tile_depletion
):
```

**Метаболизм (строки ~84-88):**
```python
# Было:
if diet_type[i] == 3:
    plant_metab = 0.4
    if food_regen_mult < 0.5:
        plant_metab *= 0.3      # hardcoded
    cost *= plant_metab

# Стало:
if diet_type[i] == 3:
    plant_metab = 0.4
    if food_regen_mult < 0.5:
        plant_metab *= plant_night_dormancy    # 0.6

```

**Фотосинтез (строки ~101-111):**
```python
# Было:
if diet_type[i] == 3:
    photo = photosynth[i] * food_regen_mult    # использует глобальный frm
    tt = tile_types[iy, ix]
    if tt == 2:
        photo *= 1.3
    elif tt == 3:
        photo *= 0.3
    photo *= 0.8 * dt
    energy[i] += photo

# Стало:
if diet_type[i] == 3:
    if food_regen_mult < 0.5:
        photo_mult = plant_photo_night_mult    # 0.1 ночью
    else:
        photo_mult = food_regen_mult           # 1.0 днём
    photo = photosynth[i] * photo_mult
    tt = tile_types[iy, ix]
    if tt == 2:
        photo *= 1.3
    elif tt == 3:
        photo *= 0.3
    photo *= 0.8 * dt
    energy[i] += photo
```
